# 🛒 Customer Purchase Prediction using Machine Learning

> DSBDA (Data Science and Big Data Analytics) Mini Project  
> Linear Regression | Python | Streamlit | scikit-learn

---

## 📌 Description

This project predicts the **Purchase Amount** a customer is likely to spend based on
their demographic and behavioural profile.  
It demonstrates a complete **end-to-end Machine Learning pipeline** — from data
generation and preprocessing through model training, evaluation, and deployment via
an interactive Streamlit web application.

---

## 🧰 Tech Stack

| Layer | Tools / Libraries |
|---|---|
| Language | Python 3.x |
| Data Manipulation | pandas, numpy |
| Visualisation | matplotlib, seaborn |
| ML Framework | scikit-learn |
| Model Persistence | pickle |
| Web App | Streamlit |

---

## 📁 Project Structure

```
customer-purchase-ml/
│
├── app.py              ← Streamlit web application
├── model.py            ← Full ML pipeline (train + evaluate + save)
├── dataset.csv         ← Synthetic customer dataset (200 rows)
├── model.pkl           ← Trained model + scaler (generated by model.py)
├── requirements.txt    ← Python dependencies
└── README.md           ← Project documentation
```

---

## 📊 Dataset Features

| Column | Description |
|---|---|
| Age | Customer age in years |
| Annual_Income | Yearly income (₹) |
| Spending_Score | Store-assigned score (1–100) |
| Years_as_Customer | How long the customer has been registered |
| Number_of_Purchases | Total purchases made |
| **Purchase_Amount** | **Target variable – amount spent (₹)** |

## 🚀 How to Run

### 1. Clone the repository
```bash
git clone https://github.com/<your-username>/customer-purchase-ml.git
cd customer-purchase-ml
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Train the model (generates model.pkl)
```bash
python model.py
```

### 4. Launch the Streamlit app
```bash
streamlit run app.py
```

The app opens automatically at **http://localhost:8501**
